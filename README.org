#+TITLE: ConwayGOL

* About

Conway's Game of Life implementation on shell script.

* Usage

one step execution.

#+BEGIN_SRC 
bin/conwaygol example-pattern/glider 
#+END_SRC

2 step execution.

#+BEGIN_SRC 
bin/conwaygol example-pattern/glider	|
bin/conwaygol
#+END_SRC
